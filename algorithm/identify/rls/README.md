# RLS — 递归最小二乘在线辨识

## 概述

RLS（Recursive Least Squares）是一种在线参数估计算法。它通过递推方式不断更新模型参数，使模型输出与实测输出的指数加权误差平方和最小。相比 LMS，RLS 收敛速度快一到两个数量级，适合电机参数这类需要快速跟踪变化的场景。

## 算法原理

### 问题定义

对于线性系统：

```
y(n) = w₁·x₁(n) + w₂·x₂(n) + ... + wₘ·xₘ(n) = Wᵀ · X(n)
```

其中 `W = [w₁, w₂, ..., wₘ]ᵀ` 是待辨识的权重向量，`X(n) = [x₁(n), x₂(n), ..., xₘ(n)]ᵀ` 是第 n 步的输入。已知观测 `X(n)` 和期望输出 `y(n)`，要求在线更新 `W`，使模型输出逼近真实值。

### 递推方程

每一步的更新由五组方程组成：

```
① 模型输出:      u(n) = Wᵀ(n-1) · X(n)
② 误差:          e(n) = y(n) - u(n)
③ 卡尔曼增益:    K(n) = P(n-1) · X(n) / [λ + Xᵀ(n) · P(n-1) · X(n)]
④ 权重更新:      W(n) = W(n-1) + K(n) · e(n)
⑤ 协方差更新:    P(n) = [P(n-1) - K(n) · Xᵀ(n) · P(n-1)] / λ
```

**各变量含义：**

| 符号 | 维度 | 说明 |
|------|------|------|
| X(n) | m×1 | 当前输入向量 |
| y(n) | 标量 | 当前期望输出 |
| u(n) | 标量 | 当前模型估计 |
| e(n) | 标量 | 估计误差 |
| W(n) | m×1 | 权重向量（待辨识参数） |
| K(n) | m×1 | 卡尔曼增益向量 |
| P(n) | m×m | 协方差矩阵（表示参数估计的不确定性） |
| λ | 标量 | 遗忘因子，0 < λ ≤ 1 |

### 遗忘因子 λ 的作用

- **λ = 1**：所有历史数据权重相等，参数收敛后不再变化（适合时不变系统）
- **λ < 1**：指数加权，越旧的数据权重越小，算法能跟踪参数变化（适合时变系统）

典型值：0.99 ~ 0.99999。越接近 1，收敛越慢但越稳定；越小，跟踪越快但对噪声越敏感。

### 协方差矩阵 P 的作用

- P 衡量参数估计的不确定性
- 初始值 `P_init` 越大，算法对初始参数的置信度越低，前期收敛越快
- 随着观测增多，P 逐渐减小，参数趋于稳定
- λ < 1 时 P 不会衰减到 0，保留了持续跟踪能力

P_init 选取建议：
- 信号可信（SNR 高）：P_init = 1 ~ 10
- 噪声较大：P_init = 100 ~ 1000

## 输入输出规范

RLS 的权重存储方式为列优先：

```
w_[x_size × y_size]  权重矩阵
  列 j 对应第 j 个输出
  行 i 对应第 i 个输入
```

当 y_size = 1（单输出）时，`w_[i]` 就是第 i 个输入的权重。

## 使用方式

### 实例化

```cpp
// 构造函数: RLS(x_size, y_size, lambda, p_init)
// 2 输入 1 输出，遗忘因子 0.99999，初始协方差 1e-5
alg::rls::RLS filter(2, 1, 0.99999f, 1e-5f);
```

### 设置初始权重

```cpp
float w[2] = { 1.453e-7f, 1.453e-7f };
filter.SetWeights(w);
```

不调用 SetWeights 时权重从 0 开始。

### 每步迭代

```cpp
float input[2]  = { torque2, omega2 };    // 输入
float desired   = measuredPower;           // 期望输出

filter.Update(input, &desired);

float* weights = filter.GetWeights();      // 辨识结果
float* output  = filter.GetOutput();       // 模型估计 u(n)
float* error   = filter.GetError();        // 估计误差 e(n)
```

## 典型参数

| 场景 | m | λ | P_init | 说明 |
|------|---|----|--------|------|
| 电机铜损/铁损辨识 | 2 | 0.99999 | 1e-5 | 慢变参数，高遗忘，小初始协方差 |
| 负载惯量辨识 | 1 | 0.95 | 100 | 变化较快，低遗忘，大初始协方差 |
| 传感器偏置校准 | 1 | 0.999 | 1 | 慢变偏置，中等遗忘 |

## 注意事项

1. **输入缩放**：RLS 对输入幅值敏感。输入 x 的尺度差异过大会导致协方差矩阵病态。建议在喂入前对输入做归一化或确保各维度量级接近。
2. **模型输出 u**：内部先算 `u = Wᵀ · X` 再算 `e = y - u`。这是完整的 RLS 流程，不要手动补 u。
3. **多输出**：支持 y_size > 1，此时权重矩阵为 `x_size × y_size`。PowerCtrl 使用单输出。
4. **拷贝**：RLS 禁止拷贝（内部持有 raw 数组），传引用使用。

# Session: 2026-05-16 TFLM 学习 + 模型训练部署

## What We Are Building

完整跑通 AI 边缘算法流程：PC 训练神经网络 → 导出 TFLite → 部署到 HPM6E00EVK 板子上推理。

## What WORKED

- **用户亲手训练了 sin 模型并部署到板子** — Python 脚本 `train_sin.py`，64→64→1 网络，loss 0.00019，烧录后预测结果正确
- **模型数据文件按文件夹组织** — `models/hello_world/` 已删除，`models/mysin/` 为当前模型
- **TFLM 线程`g_mysin`单例模式修复** — 解决了 `AddBuiltin same op twice` 和 `Model allocation started` 重复分配 bug
- **`HW_STACK_PROTECTION` 移到 HPM board conf** — `prj.conf` 删掉，`hpm6e00evk.conf` 加上，STM32F4 编译不再警告
- **`protocol_base.hpp` 清理** — 移除了 TFLM 平滑模型代码和残余 `#endif`
- **`trd_tflm.cpp` 清理** — 移除了 SIN/SMOOTH ifdef，只保留 predict 测试
- **`tflm.hpp` 改为 `TFLM_MODEL_MYSIN`** — SIN 相关代码清理完毕
- **用户学会了**：NumPy 基础（linspace, sin, zeros, randn, 切片），网络结构（Sequential, Dense, compile, fit），TFLite 转换

## What Did NOT Work (and why)

- **sed 修改文件** — 导致 System_startup.cpp 和 protocol_base.hpp 产生垃圾代码和 `#endif` 残留。后续改用 Edit 工具
